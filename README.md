# Shimizumi_Yuzuru-DiscordBot

## Run Project
1. `npm install`
1. create a file `auth.json` [TOKEN](https://discordapp.com/developers/applications/me)
	```JSON
	{
	  "token": "YOUR DISCORD BOT TOKEN"
	}
	```
2. `npm start`
